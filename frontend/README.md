# AI TripSmith Frontend

Next.js 15 + React 19 기반 여행 계획 생성 프론트엔드

## 기술 스택

- Next.js 15
- React 19
- TypeScript 5.x
- Tailwind CSS
- Mapbox GL JS

## 설치 및 실행

```bash
# 의존성 설치
npm install
# 또는
pnpm install

# 환경 변수 설정
cp .env.local.example .env.local
# .env.local 파일을 편집하여 필요한 값 설정

# 개발 서버 실행
npm run dev
# 또는
pnpm dev
```

개발 서버: http://localhost:3000

## 빌드

```bash
npm run build
npm start
```
